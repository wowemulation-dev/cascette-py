# Utility Scripts

Helper scripts for data collection and analysis.

## Scripts

### import_missing_builds.py

Searches for and imports missing builds from Wago.tools.

Usage:

```bash
python scripts/import_missing_builds.py
```

### scrape_wiki_builds.py

Scrapes build information from WoW wiki pages.

Usage:

```bash
python scripts/scrape_wiki_builds.py
```

## Note

These scripts are development utilities and not part of the main
cascette-tools package.
